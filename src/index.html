Visitor Counter

This project showcases cloud skills using AWS, Terraform, and GitHub Actions. It features a static website on AWS S3 with a visitor counter powered by Lambda, API Gateway, DynamoDB, and a CI/CD pipeline with GitHub Actions and Terraform.

BreadBasket

Breadbasket is a full-stack app using React, Node.js/Express, MongoDB, and Firebase for user authentication. Vendors can log in and place orders, while the backend automates order processing, generates reports, and emails them to recipients.

  • GitHub Repo

Match Predictor

This Python-based app uses a linear regression model to predict potential matches in dating apps based on factors like gender, age, income, and attractiveness. Users input their details, and the model provides personalized predictions of match success.
